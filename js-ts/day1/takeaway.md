## Day 001 — Execution Context, Hoisting, TDZ

- **During creation and execution phase, JS engine sees everything as either a variable or a function — there is no third category**

- **Parameters are just variables — they follow the same two phase rule**

- **Outer reference points to a context, not a variable**

- **Function EC is created only at the moment of the call, not before**
