# Conditionals — All 4 Types + Mixed

---

## My Decision Tree

```
100% always true? → Type 0
Possible and likely? → Type 1
Imaginary/unlikely now? → Type 2
Didn't happen in past? → Type 3
Past condition + present result? → Mixed
```

---

## Type 0 — Always True

**My definition:** 100% sure — no possibility of doubt

**Structure:** If + Simple Present, Simple Present

**My examples:**

1. If it rains, there is a power cut in the city
2. If you heat water, it boils
3. If police catch you, it means you broke the law
4. If you trust God, you get success

---

## Type 1 — Real Future Possibility

**My definition:** May or may not happen — second action depends on first

**Structure:** If + Simple Present, will + base verb

**My examples:**

1. If I study hard, I will join a product based company
2. If you work hard, you will pass
3. If you come home early, you will get sweets
4. If you attend the function tomorrow, you will meet him
5. If I select you, I will make you work for 10 hours

---

## Type 2 — Imaginary Perception for Future

**My definition:** Dream world — not true now, unlikely to happen

**Structure:** If + Simple Past, would + base verb

**Note:** Always use "were" not "was" for all subjects in Type 2

**My examples:**

1. If I were rich, I would buy a Tesla
2. If everybody worked hard, we would be the top company
3. If I dated you, I would marry you
4. If I had kids, I would retire early
5. If I had a family business, I would live a luxurious life
6. If I were the Prime Minister, I would make education free

---

## Type 3 — Regret for the Past

**My definition:** Past regret — already too late, didn't happen

**Structure:** If + Past Perfect, would have + past participle

**My examples:**

1. If I had completed my education on time, I would have earned more money
2. If I had married a rich girl, I would have become rich
3. If I had not left the first company, I would have become manager by 2023
4. If I had worked hard, I would have passed

---

## Mixed Conditional — Past condition + Present result

**Structure:** If + Past Perfect, would + base verb (no "have")

**My examples:**

1. If I had married in my twenties, I would be a father by now

---

## Special Conditionals — Stylistic variations of Type 1

| Special           | Meaning                         | Example                                 |
| ----------------- | ------------------------------- | --------------------------------------- |
| **Unless**        | If not                          | Unless you study, you will fail         |
| **As long as**    | On condition that               | I will help as long as you are honest   |
| **Provided that** | On condition that               | I will join provided the salary is good |
| **In case**       | Precaution                      | Take umbrella in case it rains          |
| **Even if**       | Condition doesn't change result | Even if it rains, I will go to office   |

---

## Key differences

| Type  | Tense in "if"  | Result          | Meaning                        |
| ----- | -------------- | --------------- | ------------------------------ |
| 0     | Simple Present | Simple Present  | Always true                    |
| 1     | Simple Present | will + verb     | Real possibility               |
| 2     | Simple Past    | would + verb    | Imaginary                      |
| 3     | Past Perfect   | would have + pp | Past regret                    |
| Mixed | Past Perfect   | would + verb    | Past condition, present result |

---

## Common mistakes

- _"would not have shouted"_ not _"would have not shouted"_ — "not" always after "would"
- _"If + will"_ ❌ — never use will in the if clause for Type 1
- _"If I was you"_ ❌ → _"If I were you"_ ✅ — always "were" in Type 2
