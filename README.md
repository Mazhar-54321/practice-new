# English Mastery — 24 Elements

---

| #   | Element                     | Description                                                        |
| --- | --------------------------- | ------------------------------------------------------------------ |
| 1   | **Vocabulary**              | Words and their meanings                                           |
| 2   | **Grammar Tenses**          | Present, past, future, perfect                                     |
| 3   | **Articles**                | a, an, the — when to use which                                     |
| 4   | **Prepositions**            | in, on, at, by, for, with — placement words                        |
| 5   | **Phrasal Verbs**           | give up, look into, carry on — verb + particle combinations        |
| 6   | **Connectors**              | however, therefore, although, moreover — linking ideas             |
| 7   | **Idioms**                  | bite the bullet, under the weather — figurative expressions        |
| 8   | **Collocations**            | Words that naturally go together — make a decision, do homework    |
| 9   | **Sentence Structure**      | SVO, complex sentences, questions                                  |
| 10  | **Determiners**             | a, an, the, this, that, some, any                                  |
| 11  | **Modal Verbs**             | can, could, should, must, might — ability, possibility, obligation |
| 12  | **Active vs Passive Voice** | I wrote the code vs the code was written                           |
| 13  | **Conditionals**            | if/then structures — If I had known, I would have done it          |
| 14  | **Reported Speech**         | He said that he was working                                        |
| 15  | **Question Tags**           | You are coming, aren't you?                                        |
| 16  | **Gerunds vs Infinitives**  | I enjoy coding vs I want to code                                   |
| 17  | **Pronouns**                | he, she, they, it, whose, whom                                     |
| 18  | **Adjectives & Adverbs**    | good vs well, quick vs quickly                                     |
| 19  | **Punctuation**             | commas, semicolons, apostrophes                                    |
| 20  | **Word Formation**          | prefixes, suffixes — un+happy, care+less                           |
| 21  | **Homophones**              | Words that sound same but different meaning — there/their/they're  |
| 22  | **Synonyms & Antonyms**     | big/large, happy/sad                                               |
| 23  | **Compound Sentences**      | Joining two ideas with and, but, or, so                            |
| 24  | **Interjections**           | oh, wow, well, hmm — natural conversation fillers                  |
