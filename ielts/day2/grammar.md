## Rule — Subject + Verb + Object (SVO)

The most fundamental rule of English. Verb always comes right after subject — never at the end.

| Language | Pattern                 | Example                      |
| -------- | ----------------------- | ---------------------------- |
| Urdu     | Subject + Object + Verb | Main ghar pe kaam karta hoon |
| English  | Subject + Verb + Object | I work from home             |

---

## Simple Present Tense — for daily habits and facts

```
I / You / We / They  +  verb         +  object
He / She / It        +  verb + s/es  +  object
```

| Subject | Example                  |
| ------- | ------------------------ |
| I       | I build user interfaces  |
| I       | I work from home         |
| I       | I wake up at 7am         |
| He      | He reviews my code daily |
| She     | She manages the project  |

### Negative form

```
I / You / We / They  +  don't   +  verb  +  object
He / She / It        +  doesn't +  verb  +  object
```

| Example                     |
| --------------------------- |
| I don't enjoy long meetings |
| He doesn't write unit tests |

### Question form

```
Do    +  I / You / We / They  +  verb  +  object?
Does  +  He / She / It        +  verb  +  object?
```

| Example                |
| ---------------------- |
| Do you use TypeScript? |
| Does he work remotely? |
