# Day 003 — Grammar: Future Tense

**Date:** 22-03-2026

---

## Rule — 3 Types of Future Tense

### Type 1 — Will (spontaneous decisions, predictions, promises)

```
Subject + will + base verb + object
```

| Example                   | Why                  |
| ------------------------- | -------------------- |
| I will achieve my goals   | promise to yourself  |
| It will rain tomorrow     | prediction           |
| I will help you with that | spontaneous decision |

### Type 2 — Going to (planned intentions)

```
Subject + am/is/are + going to + base verb + object
```

| Example                                | Why                   |
| -------------------------------------- | --------------------- |
| I am going to study JS internals today | already planned       |
| She is going to join a new company     | decision already made |
| We are going to deploy next Friday     | planned in advance    |

### Type 3 — Present Continuous for future (fixed arrangements)

```
Subject + am/is/are + verb(ing) + time expression
```

| Example                                  | Why               |
| ---------------------------------------- | ----------------- |
| I am meeting my manager tomorrow         | fixed appointment |
| We are deploying on Monday               | scheduled         |
| I am starting the Next.js module tonight | arranged          |

---

## Key Rule

> For goals and future plans — always use **going to** or **present continuous** — sounds more planned and serious than will.
